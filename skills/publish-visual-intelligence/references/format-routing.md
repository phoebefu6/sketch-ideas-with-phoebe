# Visual format routing

Select the smallest format that expresses the idea clearly.

| Format | Use when | Avoid when |
|---|---|---|
| Infographic | Several connected ideas form one explanatory system or journey | One claim is enough |
| Chart | Sourced numeric data supports one precise comparison or trend | Numbers are invented or incidental |
| Comic | A behavior, misconception, or sequence needs characters and timing | The lesson is purely reference material |
| Poster | One strong metaphor can carry the entire idea | Multiple claims need equal weight |
| Carousel | A concept needs 5–10 paced steps or slides | A single image remains legible |
| Card | One sharp insight, quote, or definition deserves a shareable tile | Context is necessary to avoid distortion |
| Cheatsheet | Durable frameworks, taxonomies, checklists, or comparisons need scanning | The source is mainly narrative |
| Ironic graph | Satire exposes a data/workplace contradiction and the exaggeration is clear | The audience may mistake it for real data |
| Illustration | A scene, environment, or character metaphor explains the idea | Exact quantitative structure matters |
| Logo | The task is identity or naming for a data/AI concept | The source is editorial content |
| Style study | The user explicitly wants to learn observable visual mechanics from a safe reference | It would imitate a living artist, brand, or copyrighted composition |

## Tie-breakers

1. Prefer `Chart` when trustworthy quantitative data is central.
2. Prefer `Cheatsheet` when future retrieval matters more than narrative.
3. Prefer `Infographic` when relationships matter more than sequence.
4. Prefer `Carousel` when density would make one image unreadable.
5. Prefer `Poster` or `Card` when one idea dominates.

Record the reason in `Visual Direction`.
